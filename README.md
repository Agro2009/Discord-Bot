# Discord Bot

A simple Discord bot using [discord.js](https://discord.js.org/).

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Copy the example environment file and add your bot token:
   ```bash
   cp .env.example .env
   ```

3. Run the bot:
   ```bash
   npm start
   ```

## Usage

- Type `!ping` in your Discord server to test the bot.
